## Prereqs
### Install Vagrant
```
brew install --cask vagrant
```

### Install Virtualbox
[Virtualbox's website](https://virtualbox.org)


Vagrantfile was created with 
```
vagrant init geerlingguy/centos7
```

To start the VM
```
vagrant up
```

To shutdown the VM
```
vagrant halt
```

To Delete the VM
```
vagrant destroy
```

To ssh into the VM
```
vagrant ssh
```

To see ssh info
```
vagrant ssh-config
```
